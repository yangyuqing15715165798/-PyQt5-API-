# WeatherQuery-Qt
基于 PyQt5 和和风天气 API 的桌面天气查询工具

![image](https://github.com/user-attachments/assets/12b6aa29-3795-4296-a529-12358431116a)


## 功能特性
- 实时天气展示（温度、湿度、风速、紫外线指数等）
- 未来三天天气预报（温度范围、天气状况）
- 友好的图形界面（GUI）
- 错误提示与输入校验
- 多线程请求（避免界面卡顿）

## 环境要求
- Python 3.6+
- PyQt5 5.15+
- requests 2.28+
